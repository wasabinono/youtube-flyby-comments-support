# 🪽 Fly-by Comments for YouTube - Support Hub

Welcome to the official support and feedback repository for the Fly-by Comments for YouTube Chrome Extension!

If you haven't installed the extension yet, you can get it directly from the Chrome Web Store (Insert your link here).

🐞 How to Report a Bug

If the extension isn't working on a specific YouTube video, or if you've run into a visual glitch, we want to fix it!

Please click the Issues tab at the top of this page and click New Issue. When reporting a bug, please include:

1. What browser and operating system you are using (e.g., Chrome on Mac).
2. The exact URL of the YouTube video where the issue happened.
3. A brief description or screenshot of what went wrong.
   
💡 How to Request a Feature

Have a great idea for a new setting? We are constantly looking to improve the extension!

Click the Issues tab, click New Issue, and describe your idea. Whether it's a new UI feature, a different color scheme, or a behavior change, let us know how it would improve your viewing experience.

🔒 Privacy
Your privacy is our priority. This extension runs 100% locally on your machine and never transmits or collects your personal data or browsing history.
